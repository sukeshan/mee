Identifying Tablet Purpose

Project Purpose

One day my mom gave a bunch of tablets and asked me to find the purpose of each tablet. I tried to teach my mom to find the purpose of tablets by herself using Google, but she was a little confused. This motivated me to create an app to find the uses of the tablet by using only images.

Using Spatial Information

I found an insight when I was collecting tablet images: every company prints the drug name in bold font, and the font size is bigger than other words. First I extract the text and its coordinates from the image using OCR and I use three conditions to identify the drug names:

  • Text Size
  • Keyword Technique
  • Pixel Information

Text Size

We know the drug names are printed in bold and the font size is big compared to others so I wrote conditions to filter the bold and big-size font.
